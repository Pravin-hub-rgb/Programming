Binary Search Trees

  • It is type of binary tree.
    • Value of all nodes of the left subtree are lesser than the node itself (root node).
    • Value of all nodes of the right subtree are greater than the node itself.
    • Left and RIght subtrees are also binary search trees.
    • There are no duplicate nodes.
