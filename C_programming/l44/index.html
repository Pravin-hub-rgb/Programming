Travel agency manager exercise

you manage a travel agency and you want your 'n' drivers to input (or you can take n=3 for simplicity) their following details:
1. Name
2.Driving License No
3. Route
4. Kms
Your program should be able to take n as input and your drivers will start inputting their details one by one.
Your program should print details of the drivers in a beautiful fashion.
Use structures.
