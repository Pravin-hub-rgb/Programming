Memory leak

What is memory leak in C?

  • Dynamic Memory Allocation is a way in which the size of a data structure can be changed during the runtime.
  • Memory assigned to a program in a typical architecture can be broken down into four segments:
    • Code
    • Static/global variable
    • Stack
    • Heap

What causes memory leak?

  • With great power comes great reponsibility.
  • Memory leak is caused when we don't use dynmic memory properly.
  • When we keep on allocating memory in the heap with freeing, the overall memory usage keeps on increasing.
  • This situation is the cause on memory leak i.e. the creates a memory block in the memory and forgets to delete it.
  • To avoid these memory leak situations, memory allocated on heap should always be free when not needed.
