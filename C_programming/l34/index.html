C Programming

Strings

Is string a data type in C?

  • No
  • We have char, int, float and other data types but not 'string' data type in C
  • String is a supported data type in C but it is a very useful concept used to model real world entities like name, city etc.
  • We express strings using an array of characters terminated by a null character ('\0')
    • Null character tells us that the string is terminated or ended
    • To store a string the length of character should be the length of string + 1 to store the null character ("harryare" length should be 9)

What are strings

  • String: array of characters terminated by NULL character
  • Strings in c is created by creating an array of characters
  • We need an extra character ('\0' or null character) to tell the compiler that the string ends here.

Creating string

  • We can create a character array in the following ways:
    • char name[] = "harry"; //the compiler automatically puts null character
    • char name[]= {'h','a','r','r','y','\0'}; //we have to manually put null character here
