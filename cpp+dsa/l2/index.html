How is data stored in memory?

Eg → int a = 8; // int takes 4 bytes = 32 bits
In binary, 8 = 1000 (4bits needed)
∴ int a → 00000000 00000000 00000000 00001000 → 32 bits

How are negative numbers stored?

  • The first bit tells us whether the number is positive or negative.
    • First Bit → 0 means Positive
    • 1 means Negative.
  • Steps to store -5 in binary format ↓
    1. Ignore the -ve sign (5)
    2. Write the binary representation of 5
    3. Take its 2's complement and store it.
  • Example ↓ a = -5
    1. -5 → 5 (Ignore the -ve sign)
    2. 5 → 0101 → 00....0101 (Binary)
    3. 2's complement = 1's complement + 1.
      • 5 → 00....0101
      • 1's complement → 11....1010 (Flip the bits)
      • 2's complement → 11....1011
  • Displaying Negaitve Number ↓
    1. Take 2's complement of the stored number
      • stored → 11.....1011
        • First bit shows negative
      • 1's complement 00...0100
      • 2's complement 00....0101 = 5
      • Why 2's complement ?
        • If we stored numbers as it is without using 2's complement, then representing 0 → 00.....0000 and -0 → 10....000 will be equal and thus waste of space.

Store only Positive Integers

The default signed representation allows us to store both positive and negative values.
To store only positive integers, we use unsigned.
Eg → unsigned int a = 10;

What if we store a negative value in an unsigned number?
Example → unsigned int a = -112;
cout << a << endl;
output → 4294967184?

  • Explaination
    • Unsigned int uses all 32 bits to store the value and the MSB (=1) will make the value.
    • An unsigned int does not use the 2's complement again to display the number.
    • Thus, 11...10010000. gets printed as it is in decimal.

Logical Operators

&&, ||, !

  1. Logical AND ↓
    • All conditions must be true for the output to be true
    • Example ↓ 
                                           
int a = 5, b = 10, c = 15;
cout << ((a > 0) && (b != 0) && (c <= 15)); output < 1
  2. Logical OR ↓
    • At least 1 condition must be true for the output to be true.
    • Example ↓ 
                                   
int a = 5, b = 10, c = 15;
cout << ((a>5) || (b<10) || (c >= 15)); output → 1
  3. Logical NOT ↓
    • Inverts the logic. true ⟺ False, non-zero ⟺ zero
    • Example ↓ 
                                   
int a = 5, b = 10, c = 15;
cout <<(!a)<<endl; output → 0
cout <<(!b)<<endl; output → 1
