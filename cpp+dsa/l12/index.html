Binary Search

This topic is very important, according to online test and interviews.

  • Condtion to apply binary search
    • Element should be in monotonic function
      • Increasing or decreasing order
  • Steps to follow
    1. Find mid element of the array and check whether it is equals to the key element, if it is return the index.
      • (upper bound + lower bound ) / 2
        • Optimized → lower bound + (upper bound - lower bound) / 2
        • So that we don't exceed the range on integer
    2. check if it is lesser than the mid
      • If yes update the searching space
      • put upper bound to mid - 1
    3. check if it is greater than the mid
      • If yes update the searching space
      • put lower bound to mid + 1;
                

                    #include <iostream>
                        using namespace std;
                        
                        int binarySearch(int arr[], int size, int key)
                        {
                            int start = 0;
                            int end = size - 1;
                        
                            while (start <= end)
                            {
                                // int mid = (start + end) / 2; locially its correct but
                                // we know that max int can be 2^31 - 1
                                // if start and end are 2^31 - 1 then this will not be in int range
                                // so we do this
                                int mid = start + (end - start) / 2;
                                if (arr[mid] == key)
                                {
                                    return mid;
                                }
                                if (key > arr[mid])
                                {
                                    start = mid + 1;
                                }
                                else
                                {
                                    end = mid - 1;
                                }
                                return mid;
                            }
                        }
                        
                        int main()
                        {
                            int even[] = {2, 4, 6, 8, 10, 12, 18};
                            int odd[] = {3, 5, 7, 9, 11, 15, 17};
                            cout << "Index of 6 is " << binarySearch(even, sizeof(even) / sizeof(int), 6);
                            return 0;
                        }

Comparing it with linear search

  • Linear search for array of 1000 size
    • In worst case the number of comparisions will be 1000
    • Time complexity → O(n)
  • Binary search in an array of 1000 size
    • Size ↓
      • 1000 → 500 → 250 → 125 → 64 → 32 → 16 → 8 → 4 → 2 → 1
      • 10 comparisons only
    • Time complexity → O(log n)
