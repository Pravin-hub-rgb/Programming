STL

  • Standard temporary library
  • The Standard Template Library (STL) is a set of C++ template classes to provide common programming data structures and functions such as vector, lists, stacks, etc.
  • Knowledge of STL is very essential because
    • We don't have to write code from scratch
    • Example → Instead of writing whole code of binary search we can use STL which will do it in a single line.
    • So instead of making our own custom data structure we can use it.

Deque geeksforgeeks reference

  • Here we can perform push, pop, deletion and insertion operation in both front and end
  • Its dynamic → size can be increased
                            

                                // include its library 
                                #include<deque> 

                                int main() {

                                    // declaration 
                                    deque<int> d; 

                                    // insertion 
                                    d.push_back(1);
                                    d.push_front(2);

                                    // finding size 
                                    d.size();

                                    // deletion from back 
                                    d.pop_back();
                                    // deletion from front 
                                    d.pop.front(); 

                                    // accessing first index element 
                                    d.at(1);

                                    // getting first and last element 
                                    d.front(); d.back();

                                    // checking if it is empty 
                                    d.empty();

                                    // deleting first 2 element 
                                    d.erase(d.begin(), d.begin()+2);

                                }

Priority queue geeksforgeeks reference

  • It is a type of queue in which the first element is always greatest 
                            

                                #include<queue> 

                                int main() {

                                    // max heap
                                    priority_queue<int> maxi;

                                    // min heap 
                                    priority_queue<int, vector<int>, greater<int> > mini;

                                    // inserting elements 
                                    maxi.push(1);
                                    maxi.push(5);
                                    maxi.push(4);

                                    // extracting elements 
                                    int n = maxi.size();
                                    for(int i = 0; i < n; i++){
                                        cout<<maxi.top()<<" ";
                                        maxi.pop();
                                    }
                                    // above will print largest element then it will pop it so it can print the next largest element 

                                    // same with min priority queue 

                                    // checking if it is empty 
                                    mini.empty();
                                }

Algorithms

The header algorithm defines a collection of functions specially designed to be used on a range of elements. They act on containers and provide means for various operations for the contents of the containers.

Binary Search || upper/lower bound

                            

                                #include<algorithm> 
                                #include<vector> 

                                int main(){

                                    vector<int> v;
                                    
                                    v.push_back(1);
                                    v.push_back(3);
                                    v.push_back(6);
                                    v.push_back(7);
                                    // binary search works in sorted array 

                                    binary_search(v.begin(), v.end(), 6); // finding 6  

                                    // finding lower bound
                                    .lower_bound(v.begin(), v.end(), 6);
                                    // finding upper bound 
                                    .upper_bound(v.begin(), v.end(), 6);
                                }
