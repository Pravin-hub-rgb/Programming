If-else || while loop

If statements

                

                    if(condition){
                        //code
                    }

If the condition within () is true, then execute the entire code block within {}.

                
                    
                    if(a>b){
                        cout<< a << endl;
                    }
                    if(a<b){
                        cout<< b << endl;
                    }

In the above example, instead of checking again for b>a in the second if block, we know that if if the first if condition does not get fulfilled then the second block must be executed no matter what. This can be achieved using on if-else block. 

                
                    
                    if(a>b){
                        cout<< a << endl;
                    }
                    else{
                        cout<< b << endl;
                    }
