Basic Input/Output & More

  • C++ language comes with different librariers, which helps us in performing input/output operations.
  • In C++ sequence of byters corresponding to the input and out are commonly known as streams.
  • There are two types of streams:
    • Input stream
      • In the input stream, the direction of the flow of bytes occurs from the input device (for example keyboard) to the main memory.
    • Output stream
      • In output stream, the direction of flow of byters occurs from main memory to the output device (for example display)
                

                        #include <iostream>
                        using namespace std;
                        int main()
                        {
                            int num1, num2;
                            cout << "Enter the value of num1: \n"; // '<<' is called insertion operator
                            cin >> num1;                           // '>>' is called extraction operator
                        
                            cout << "Enter the value of num2: \n";  // 'cout' keyword is used to print
                            cin >> num2; // 'cin' keyword is used to take input at run time
                        
                            cout << "The sum is " << num1 + num2;
                            return 0;
                        }

Reserved keywords in C++

Reserved keywords are those keywrods that are used by the language itself, which is why these keywords are not availabe for re-definition or overloading.
In short, you cannot create variables with these names.
