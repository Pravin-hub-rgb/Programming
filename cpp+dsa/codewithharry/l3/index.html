Variables & comments in C++

Variables in C++

  • Variables are containers to store our data.

Variables are of different types, for example

  1. int → int is used to store integer data e.g- (-1, 2, 5, -9, 3, 100).
  2. float → is used to store decimal numbers e.g (0.5, 1.05, 3.5, 10.5)
  3. char → is used to store a single character e.g. ('a', 'b', 'c', 'd')
  4. boolean → is used to store "true" or "false"
  5. double → is also used to store decimal numbers but has more precision than float, e.g. (10.5895758440339)

Syntax for declaring variables in C++

  1. Data_type Variable_name = value;
  2. Ex: int a = 4; char letter = 'p';
  3. Ex: int a = 4, b = 6;

Comments

  • A comment is a human-readable text in the source code, which is ignored by the compiler.
  • There are two ways to write comments.
    1. Single line comments → "//" before a single line of text to make it unparsable by the compiler
    2. Multi-line Comments → "/*" as the opening and "*/" as the closing of the comment. We then write text in betwen them.
