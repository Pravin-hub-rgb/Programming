Pointers

  • A pointer is a data type which holds the address of other data type.
  • the "&" operator is called "address off" operator
  • * operator is called "value at" dereference operator.
                

                    int a = 3;
                    int *b = &b;
                    cout<<b; // 0x34rff08
                    cout<<&a; // 0x34rff08

                    cout<<" the value at address b is "<< *b <<endl; // 3

Pointer to pointer

  • Pointer to pointer is a simple concept, in which we store the address of one pointer to another pointer. 
                    

                        int a=3;
                        int* b;
                        b = &a;

                        int **c = &b;
                        cout<<"the address of b is "<<&b<<endl; // 0x34rff08
                        cout<<"the address of b is "<<c<<endl; // 0x34rff08
                        cout<<"the value at address of c is "<<*c<<endl; // 0x34rff088
                        cout<<"the value at address value_at(value_at(c)) is "<<**c<<endl; // 3
