Structures in C++

  • The structure is a user-defined data type that is available in C++
  • Structures are used to combine different types of data types.
                

                    struct employee
                    {
                        /* data */
                        int eId; 
                        char favChar; 
                        float salary; 
                    };

Now we have created a structure now we can create instances of our structure employee.

                

                    int main() {
                        struct employee harry;
                        harry.eId = 1;
                        harry.favChar = 'c';
                        harry.salary = 120000000;
                        cout<<"The value is "<<harry.eId<<endl; 
                        cout<<"The value is "<<harry.favChar<<endl; 
                        cout<<"The value is "<<harry.salary<<endl; 
                        return 0;
                   }

Another way to create structure variables without using the keyword "struct" and the name of the struct ↓

                

                    typedef struct employee 
                    {
                        // data 
                        int eId; 
                        char favChar;
                        float salary;
                    } ep;

we have used a keyword "typedef" before struct and after the closing bracket of structure, we have written "ep".

Unions

  • Unions are similar to structures but they provide better memory management then structures.
  • Unions use shared memory so only 1 variable can be used at a time.
                    

                        union money
                        {
                            /* data */
                            int rice; //4
                            char car; //1
                            float pounds; //4
                        };
  • We can only use 1 variable at a time otherwise the compiler will give us a garbage value
  • The compiler chooses the data type which has manimum memory for the allocation.
                    

                        int main(){
                            union money m1;
                            m1.rice = 34;
                            cout<<m1.rice;
                            return 0;

                            // we have assigned values to (rice) fields of the union money 
                            // the main thing to note here is that once we have assigned a value to the union field "rice"
                            // now we cannot use other fields of the union otherwise we will get garbage value.
                    }

Enums

  • Enums are user-defined types which consist of named constants.
  • Enums are used to make the program more readable
                    

                        int main(){
                            enum Meal { breakfast, lunch, dinner};
                            Meal m1 = lunch;
                            cout<<m1;
                            return 0;
                        }
  • We have created an enum "Meal" in which we have stored three named constants (breakfast, lunch, dinner)
  • then we have assigned the value of "lunch" to the variable "m1: and at the end
  • We have printed "m1"
  • The main thing to note here is that (breakfast, lunch, dinner) are constants;
    • The value for "breakfast" is 0
    • The value for "lunch" is 1
    • The value of "dinner" is 2
