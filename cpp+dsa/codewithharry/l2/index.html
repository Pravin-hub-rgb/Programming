Basic Structure of a C++ program

                    

                        // simple program 

                        #include<iostream>

                        int main(){
                            std::cout<<"Hello World";
                            return 0;
                        }

Understanding above program

#include<iostream>

  • This whole text is called a header file
  • include is a keyword used to add libraries in our program.
  • iostream is the name of a library which helps us to get input data and show output data.
    • The iostream library also has many more uses; it is not only limited to input and output.

int main() {

  • int is a return type which is called integer and "main()" is a function, the brackets "()" denotes that it is a function.
  • The curly brace "{" indicates that is is an opening of a function, and the curly brace "}" indicates that it is the closing of a function.

std::cout<<"hello word";

  • std is a namespace, "::" is the scope resolution operator and "cout<<" is a function which is used to output data, "hello world" is out output string
  • ";" is used to end a statement
  • The code "std::cout" tells the compiler that the "cout" identifier resides in the std namespace.
  • Main point here are
    • We can write as many statements as we want in a single line, but for a neat and clean code, one statement at a line is good
    • Anything which is written between double quotation "" is a string literal

return 0

  • the return keyword will return 0 as an integer to out main function "int main()"
  • Returning 0 as a value to the main function means successful termination of the program.
