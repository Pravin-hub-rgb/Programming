Break and Continue statements

Break Statements

  • Break statements in loops are used to terminate the loop.
                

                    int main(){
                        fot(int i = 0; i < 40; i++){
                            cout<<i;
                            if(i==2){
                                break;
                            }
                        }
                    }

Continue statements

  • Continue statement only terminates the current iteration.
                

                    int main(){
                        fot(int i = 0; i < 40; i++){
                            cout<<i;
                            if(i==2){
                                continue;
                            }
                        }
                    }
