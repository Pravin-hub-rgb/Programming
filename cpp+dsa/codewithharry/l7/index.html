Reference Variables & Typecasting

Built-in Data Types

Built-in data types are pre-defined by the language and can be used directly. 

                

                    int c = 45; // global variable 

                    int main(){
                        int a, b, c;
                        cout<<"Enter the value of a: "<<endl;
                        cin>>a;
                        cout<<"Enter the value of a: "<<endl;
                        cin>>b
                        c = a + b;
                        cout<<"The sum is "<< c << endl;
                        cout<<"The global c is "<< ::c << endl;
                    }
  • To access the value of the global variable "c", we use scope resolution operator "::" with the "c" variable.

Float, Double and Long Double Literals

                    

                        #include <iostream>
                            using namespace std;
                            int main()
                            {
                                float d = 34.4F;
                                long double e = 34.3L;
                                cout << " The size of 34.4 is " << sizeof(34.4) << endl;
                                cout << " The size of 34.4f is " << sizeof(34.4f) << endl;
                                cout << " The size of 34.4F is " << sizeof(34.4F) << endl;
                                cout << " The size of 34.4l is " << sizeof(34.4l) << endl;
                                cout << " The size of 34.4L is " << sizeof(34.4L) << endl;
                                return 0;
                            }
  • Crucial concept
    • double is the default type given to a decimal literal (34.4 is double by default and not float),
    • so to use it as float, you have to specify it like "34.4F"
    • To display the size of float, double and long double literals, we have used a "sizeof" operator.

Reference Variable

  • Reference variable can be defined as another name for an already existing variable.
  • These are also called an alias.
  • For example, we have a variable with the name of "sum", but we also want to use the same variable with the name of "add", to do that we will make a reference variable with the name of "add:
                    

                        float x = 455;
                        float & y = x; // '&' is used to make it reference variable
                        // the 'y' variable will start referring to the value of the variable 'x'
                        cout<< x << endl; // 455
                        cout<< y << endl; // 455

Typecasting

Defined as converting one data type into another.

                    

                        #include <iostream>
                            using namespace std;
                            int main()
                            {
                                int a = 45;
                                float b = 45.46;
                                cout << "the value of a is " << (float)a << endl; // one way of typecast
                                cout << "the value of a is " << float(a) << endl; // another way of typecast
                            
                                cout << "the value of b is " << (int)b << endl;
                                cout << "the value of b is " << int(b) << endl;
                                int c = int(b);
                            
                                cout << "the expression is " << a + b << endl;
                                cout << "the expression is " << a + int(b) << endl;
                                cout << "the expression is " << a + (int)b << endl;
                                return 0;
                            }
