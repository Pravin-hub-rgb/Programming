Arithmetic operators

  • Arithmetic operators are used for performing mathematical operations like (+, -, *). 
                        

                            int a = 5, b = 6;
                            a + b; // will add a and b values

                            a - b // subtract 

                            a*b // multiply 

                            a/b // divide 

                            a%b // remainder 

                            a++ // will first use value of a and then increment it by one 

                            a-- // will first use value of a and then decrement it by one 

                            ++a // first increment it 

                            --a // first decrement it

                            // assignment operators 
                            // used for assigning values to variables .

                            // comparison operators 
                            // used for comparing two values 
                            a == b // will campare a and b values and check if they are equal. The output will be 1 if equal, and 0 if not 
                            a!=b 
                            a>=b 
                            a<=b 
                            a>b 
                            a<b


                            // Logical operators 
                            // logical operators are used for comparing two expressions. 
                            // for example ((a==b) && (a>b)). 

                            // && 
                            ((a==b) && (a<b)) // will first compare a and b values and check if they are equal or not; 
                            // if they are equal, the next expression will check whether "a" is smaller than "b".
                            // the output will be 1 if both expressions are true and 0 if not.

                            // ||
                            ((a==b) || (a<b)) // will first compare a and b values and check if they are equal or not,
                            // even if they are not equal it will still check the next expression ie whether "a" is smaller than "b" or not.
                            // the output will be 1 if if any one of expressions is true and 0 if both are flase.

                            (!(a==b)) // will first compare a and b values and check if they are equal or not.
