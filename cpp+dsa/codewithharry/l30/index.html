Parameterized and Default Constructors

  • Parameterized Constructors
    • Those constructors that take one or more parameters.
  • Default Constructors
    • Those constructors that take no parameters
  • The main thing to note here are athat constructors are written in the public section of the class and the constructors don't have a return type.
                    

                        #include<iostream> 
                        using namespace std;

                        class Complex
                        {
                            int a, b;                        

                        public: 
                            Complex(int, int); // constructor declaration

                            void printNumber()
                            {
                                cout << "Your number is " << a << " + " << b << "i" << endl;
                            }
                        }

                        Complex ::Complex(int x, int y) // ----> This is a paramaetrized constructor as it takes 2 parameters
                        {
                            a = x;
                            b = y;
                        }
                    
                
                    

                        int main(){
                            // Implicit call
                            Complex a(4, 6);
                            a.printNumber();
                        
                            // Explicit call
                            Complex b = Complex(5, 7);
                            b.printNumber();
                        
                            return 0;
                        }
