Bitwise operator

These all works on bit

Bitwise AND (&)

E.g: 2 and 3 = 010 & 011 = 010
Boths bits must be 1 to get 1
E.g.: 5 & 7 = 101 & 111 = 101

Bitwise OR : (|)

If at least one of the bits is 1 then the output will be 1.
2 | 5 = 010 | 101 = 111
3 | 6 = 011 | 110 = 111

Bitwise NOT (~)

E.g: consider 4 byte representation
a = 2 = 000....0010
~a = ~2 = 111....1101 (signed bit)

Bitwise XOR (^)

If different bits → 1
If same bits → 0
E.g: 5 ^ 7 = 101 ^ 111 = 010

Left Shift : (<<)

E.g: 5 << 1 → Left shit 5, 1 time
= 101 << 1 → 00.....0101 << 1
== 00....1010 = 10 = 5*21

E.g: 3 << 2 → 00...011 << 2
= 00...1100 = 12 = 32
In some cases we multiply with power of 2 but in some cases, this isn't ture.

Right Shift : (>>)

E.g: 15 >> 1 = 000...1111 >> 1
= 000...0111 = 7

E.g: 5 >> 2 = 000...0101 >> 2
000...0001 = 1
Padding in << and >> for positive numbers is done with 0.
For negative numbers, padding is compiler dependent.

Increment/Decrement

  • We can write i = i + 1 as i++ or ++i
    • i++ is post-increment
    • ++i is pre-increment
  • We can write i = i - 1 as i-- or --i
  • a = a + b is same as a += b

post-increment → the value gets used first and then increments. 

                        

                            int i = 3, a = 2;
                            int sum = a + (i++);
                            //sum = 2 + 3;
                            sum = 5;
                            // now i is 4.

pre-increment → the value gets incremented first and then gets used.

Scope

  • The life-time of a variable where does the variable exist and after what line of code will it get destroyed
  • In short, its accessibility

Example 1

                

                    #include <iostream>
                    using namespace std;

                    int main(){
                        int a = 3;
                        cout<< a << endl;

                        if(true){
                            cout<< a << endl;
                        }
                    }

a is accessible throughout the main() function, after it is declared.

Example 1

                

                    #include <iostream>
                    using namespace std;

                    int main(){
                        int a = 3;
                        cout<< a << endl;
                        // this will print 3
                        // as this is 'local' to int main()

                        if(true){
                            int a = 5;
                            cout<< a << endl;
                            // it will print 5
                            // this is 'local' to if block,
                            // and is only accessible within the if block
                        }
                    }

a is accessible throughout the main() function, after it is declared.
