Bubble Sort

  • It is called bubble sort because it bubbles up lighter elements to the left and stores larger elements towards the right. 
                    

                        void bubbleSort(int arr[], int n)
                        {
                            int i, j;
                            bool flag;
                            // Outer pass
                            for(i = 0; i < n; i++)
                            {
                                flag = false;                                   // Set flag as false
                                for(j = 0; j < n-i-1; j++)
                                {
                                    // Compare values
                                    if( arr[j] > arr[j+1])
                                    {
                                        swap(arr[j],arr[j+1]);
                                        flag = true;
                                    }
                                }
                                // If no to elements are swapped then
                                // array is sorted. Hence Break the loop.
                                if(!flag)
                                {
                                    break;
                                }
                            }
                        }
  • Time complexity
    • Best case → When the array is already sorted → O(n) (We get this when we optimize our code)
    • Worst case → When the array is sorted in descending order → O(n2)
